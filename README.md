# MineWalker

MineWalker is a game inspired by classic minesweeper but with a unique twist â€“ instead of clicking on cells, you navigate and explore the map by walking on it. Your goal is to safely traverse the area, avoiding hidden bombs, and reach the destination.

## How to Play

1. **Run the Game:**
    - Execute the script `start.sh` to launch the MineWalker game.

```
./start.sh
```

2. **Gameplay:**
    - Navigate through the map by walking on cells.
    - Avoid stepping on hidden bombs.
    - Reach the destination safely to win the game.

3. **Controls:**
    - Use arrow keys to move: Up, Down, Left, and Right.

## Description

Mine Walker is written in Go and utilizes WebAssembly (WASM) for web deployment.
